History

Everything started thanks to Dimitri’s desire for realizing an entrepreneurship related with food. This desire became an idea and then a project, with the involvement of Maylis and Kevin.

In October 2015, we started SharEat in Chile and, realized more than 40 collaborative meals using Facebook and Google Apps as platforms. After a while, we decided to start creating a website. We realized than the domain name SharEat was already used, so we changed it for the current myCommuneaty.

In April 2017, we were happy to start using our first version of the website in Melbourne. We keep working hard in this project and we would be happy to receive your feedbacks, comments or ideas. So, please feel free to contact and join us !

Let’s start the Aussie experience for myCommuneaty !
